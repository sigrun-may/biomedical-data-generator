biomedical_data_generator.meta.compute_feature_roles

biomedical_data_generator.meta.compute_feature_roles(meta)[source]

Derive the six-way generative feature-role partition from a DatasetMeta.

The partition is reconstructed purely from the structural index sets that the generator already records on meta (informative and noise indices, cluster layout, anchor columns, and per-cluster anchor properties). Each cluster’s relevance is derived from the generated signal — a class- dependent mean shift or a class-dependent within-cluster correlation (differential co-expression) — rather than read from the declared anchor_role. No feature matrix is required.

Parameters:

meta (DatasetMeta) – Resolved dataset metadata produced by biomedical_data_generator.generate_dataset().

Returns:

A FeatureRoles instance assigning every feature column to exactly one of the six generative roles, together with a column-to-cluster membership map.

Return type:

FeatureRoles
