biomedical_data_generator.generate_correlated_cluster

biomedical_data_generator.generate_correlated_cluster(n_samples, size, rho=0.7, structure='equicorrelated', random_state=None, label=None)[source]

Generate a single correlated feature cluster (no labels y involved).

Returns (X_cluster, meta) where meta contains the empirical correlation matrix.

Parameters:

  • n_samples (int) – Number of samples (rows).

  • size (int) – Number of features (columns).

  • rho (float) – Target correlation between features (0 ≤ rho < 1).

  • structure (Literal['equicorrelated', 'ar1']) – “equicorrelated” or “ar1”.

  • random_state (int | None) – Random seed for reproducibility.

  • label (str | None) – Optional didactic tag for this cluster.

Returns:

  • X (np.ndarray): Shape (n_samples, size) with standardized columns.

  • meta (dict): Metadata with keys: size, rho, structure, random_state, label, corr_matrix (size x size), mean_offdiag, min_offdiag.

Return type:

tuple

Raises:

ValueError –
