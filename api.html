API ReferenceÔÉÅ

The main public API of biomedical-data-generator. These objects are re-exported at the top level of the package and are recommended for user-facing code.

biomedical_data_generator.generate_dataset

Generate synthetic biomedical dataset with specified feature structure.

biomedical_data_generator.DatasetConfig

Configuration for synthetic dataset generation.

biomedical_data_generator.ClassConfig

Configuration for a single class in the dataset.

biomedical_data_generator.CorrClusterConfig

Correlated feature cluster simulating coordinated biomarker patterns.

biomedical_data_generator.BatchEffectsConfig

Configuration for simulating batch effects.
